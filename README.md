# NXT Everest Inventory Dashboard

## 📊 Overview
This project demonstrates the creation of a **business intelligence dashboard** for analyzing inventory returns. It highlights my ability to design a data model, build DAX measures, and visualize insights through interactive charts.

## 🔨 Process
1. **Data Modeling**  
   - Built a structured data model to handle 30K+ orders and 958 returns.  
   - Defined relationships between SKUs, warehouses, 3PL providers, and order statuses.  

2. **DAX Measures**  
   - Created custom measures to calculate key metrics such as total orders, return rate (3%), and segmentation by SKU, location, and type.  
   - Applied logic for completed vs. processed returns and monthly trends.  

3. **Visualizations**  
   - Designed clear, interactive graphs to highlight:  
     - Returns by SKU (e.g., RSX08 as highest return driver).  
     - Geographic performance by warehouse and city.  
     - 3PL provider comparison (NTW vs. TBC).  
     - Order status and return type breakdowns.  

## 🎯 Key Insights
- Overall return rate is **3%**, significantly lower than industry averages.  
- SKU RSX08 contributes the highest returns, suggesting a potential quality issue.  
- NTW handles 59% of returns, indicating reliance on one logistics partner.  
- Most returns are completed (88.8%), showing strong process closure.
